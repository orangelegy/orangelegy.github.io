英语语法框架

一、简单句

什么 + 怎么样

1. 可以独立完成的动作：不及物动词

Papa Rabbit sleeps.
兔老爹 睡觉。

主语 + （不及物动词）

2. 有一个动作的承受者：及物动词

Papa Rabbit likes you.
兔老爹 喜欢你。

主语 + （及物动词）+ 宾语

3. 有两个动作承受者：双及物动词

Papa Rabbit teaches you English
兔老爹 教 你 英语。

主语 + （双及物）动词 + 间接宾语 + 直接宾语

4. 只有一个动作承受者：混合及物动词

Papa Rabbit considers you smart.
兔老爹 认为 你 聪明。

主语 + （复杂及物）动词 + 宾语 + （宾语）补语

5. 把这个词后面的信息赋予给前者：系动词

Papa Rabbit looks tall.
兔老爹 看上去 高。

主语 + （系）动词 + （主语）补语 / 表语

句子成分

主语 Subject 谓语动词 Predicate Verb 宾语 Object 宾语补语 Object Complement

主语补语(表语) Sub Complement 定语 Attributive 状语 Adverbial 同位语 Appositive

二、复杂句

主句 + 从句

  1. 名词性从句：主语从句 宾语从句 表语从句 同位语从句
  2. 定语从句
  3. 状语从句

词类

名词 Nouns 冠词 Articles 代词 Pronouns 形容词 Adjectives 数词 Numerals

副词 Adverbs 介词 Prepositionals 叹词 interjections 连词 Conjunctions 动词 Verbs

谓语动词的“三大本领” 时 Tense 体 Aspect 气 Mood

  1. 表示动作的时间（现在、过去、将来、过去将来）
  2. 表示动作的状态（体） （一般、完成、进行、完成进行）
  3. 表示动作的假设，情感等

非谓语动词

  1. 动词不定式
  2. 现在分词
  3. 动名词
  4. 过去分词
