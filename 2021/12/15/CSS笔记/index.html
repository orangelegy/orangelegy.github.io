CSS书写规范

  • 空格规范
    • 选择器与{之间必须包含空格。
    • 属性名与之后的:之间不允许有空格，:与之后的属性之间必须包含空格。
  • 选择器规范
    • 当一个rule包含多个selector时，每个选择器声明必须独占一行。
    • 选择器的嵌套层级应不大于3级，位置靠后的限定条件应尽可能精确。

选择器

  • 标签选择器

  • 类选择器

  • ID选择器

  • 通配符选择器

    * { }

  • 伪类选择器

    • 链接伪类选择器

      :link		未访问的链接
:visited	已访问的链接
:hover		鼠标移动到链接上
:active		选定的链接（长按显示的状态）

      注意书写的时候，他们的顺序尽量不要颠倒

    • 结构伪类选择器（CSS3）

      :first-child		
	选取属于其父元素的首个子元素的指定选择器
:last-child			
	选择属于其父元素的最后一个子元素的指定选择器
:nth-child(n)		
	匹配属于其父元素的第n个子元素，不论元素的类型
:nth-last-child(n)	
	从最后一个元素开始计数，匹配属于其父元素的第n个子元素，不论元素的类型

      n可以是数字、关键字或公式

    • 目标伪类选择器

      :target		可用于选取当前活动的目标元素

  • 复合选择器

    • 交集选择器

      交集选择器由两个选择器构成，其中一个为标签选择器，第二个为class选择器，连个选择器之间不能有空格，如：

      div.red {
    
}

    • 并集选择器

      各个形式的选择器通过,隔开，如：

      a,.red,#logo {
    
}

    • 后代选择器

      又称为包含选择器，用于选择元素或元素组的后代，通常写法是把外层标签写在前面，内层标签写在后面，中间用空格分隔，如：

      .class a {

}

    • 子元素选择器

      只能选择某元素的子元素，写法就是把父级标签放在前面，子级标签放在后面，中间用>隔开，符号两边各保留一个空格。如：

      .class > a {
    
}

  • 属性选择器

    即选择某些标签带有特殊属性的选择器

    选择器 示例 含义
    E[attr] 存在attr属性即可
    E[attr=val] input[type=text] 属性值完全等于val
    E[attr*=val] 属性值里包含val字符并且在任意位置
    E[attr^=val] 属性值里包含val字符并且在开始位置
    E[attr$=val] 属性值里包含val字符并且在结束位置

  • 伪元素选择器（CSS3）

    E::first-letter		文本的第一个单词或字
E::first-line		文本的第一行
E::selection		可改变选中文本的样式

E::before
E::after	
在E元素的开始位置和结束位置创建一个元素，该元素为行内元素，且必须要结合content属性使用

字体样式

font：综合设置字体样式

选择器 {font: font-style font-weight font-size/line-height font-family}

font属性必须按照顺序书写，各个属性以空格隔开，必须保留font-size和font-family属性，其余可以省略

CSS三大特性

  • 层叠性 略

  • 继承性 略

  • 优先级

    • 特殊性（Specificity）

    关于CSS权重我们需要需要一套特殊的公式去计算，这就是CSS Specificity，我们称为CSS特性或称非凡性，它是一个衡量CSS值优先级的一个标准，具体规范如下：

    Specificity用一个四位的数字串（CSS2是三位）来表示，更像四个级别，值从左到大，一级大于一级，数位之间没有进制，级别之间不可超越。

    继承或*的贡献值 0，0，0，0
    每个元素（标签）贡献值为 0，0，0，1
    每个类，伪类贡献值为 0，0，1，0
    每个ID贡献值为 0，1，0，0
    每个行内样式贡献值为 1，0，0，0
    每个!important贡献值为 ∞无穷大
