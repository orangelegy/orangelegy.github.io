npm包管理器常用模块安装命令

2021-12-07

  • cnpm 安装（淘宝镜像）

     npm install -g cnpm -registry=https://registry.npm.taobao.org

  • Vue.js 安装（最新稳定版）

    npm install vue

  • Vue CLI 安装

    npm install -g @vue/cli

  • Axios 安装

    npm install axios

  • Mock.js 安装

    npm install mockjs

  • Element UI 安装

    npm i element-ui -S

  • ECharts 安装

    npm install echarts --save

  • Hexo 安装

    npm install hexo-cli -g
